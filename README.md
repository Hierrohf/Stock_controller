# 🧃 API de Controle de Saída de Produtos

Esta é uma API REST desenvolvida com **Spring Boot** para controle de saída de produtos em pontos de venda. Ideal para estabelecimentos que precisam rastrear o consumo em tempo real, como bares, restaurantes e eventos. A ideia nasceu no calor da rotina — e tá sendo refinada a cada linha de código escrita.

> ⚠️ **Essa API está em constante evolução.** Conforme novas tecnologias forem aprendidas, melhorias e novos recursos serão adicionados.

---

## 🚀 Objetivo

Gerenciar de forma simples e eficaz a saída de produtos por ponto de venda, funcionário e data/hora, com recursos para gerar resumos e relatórios operacionais.

---

## 📚 Tecnologias e Dependências

- Java 17+
- Spring Boot
- Spring Web
- Spring Data JPA

## ✅ Funcionalidades

- 🔄 Registro de saídas com validações
- 📊 Relatórios por produto, ponto e funcionário
- ⏰ Filtros por data e horário
- 🧾 Histórico completo de movimentações
